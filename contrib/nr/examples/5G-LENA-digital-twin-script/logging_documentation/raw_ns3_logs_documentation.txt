
Documentation for log files and the values that they log with a short description 
for each  

columns_used holds all the columns including context and metrics that are taken from 
each log file and used in creating parsed logs.   

Note on timestamps:
- time_us is the simulation time when the trace fires at the layer where it is logged.
- Rx* logs mean the PDU is delivered up into that layer (received by the layer).
- Tx* logs mean the PDU is emitted down to the next lower layer (sent by the layer).









mobility_trace.txt (recorded at UE nodes; N/A UL/DL)
- time_us: Simulation time in microseconds when the UE position sample is written.
- ue_id: UE index used in this script (derived in MakeUeTraceIds).
- imsi: UE IMSI corresponding to ue_id (from MakeUeTraceIds).
- cell_id: Serving cell ID for the UE at log time (from MakeUeTraceIds).
- rnti: UE RNTI from the UE RRC (if available).
- pos_x: UE X position in meters from MobilityModel.
- pos_y: UE Y position in meters from MobilityModel.
- pos_z: UE Z position in meters from MobilityModel.
- vel_x: UE X velocity in m/s from MobilityModel.
- vel_y: UE Y velocity in m/s from MobilityModel.
- vel_z: UE Z velocity in m/s from MobilityModel.

columns_used = []

sim_info.txt (recorded by controller script (not UE/gNB); N/A UL/DL)
- parameter: Parameter name for the run configuration snapshot.
- value: Parameter value recorded for that name.

columns_used = []

delay_trace.txt (recorded at UdpServer receiver (UE for DL, remote host for UL); UL or DL per dir)
- time_us: Receive time in microseconds at the UdpServer when the packet arrives.
- direction: Direction inferred by receiver ("UL" if remote host receives, "DL" if UE receives).
- ue_id: UE index associated with the packet (derived from context or IP).
- imsi: IMSI for the UE that sent/received the packet.
- cell_id: Serving cell ID for that UE at log time.
- rnti: UE RNTI from the UE RRC (if available).
- pkt_size: Packet size in bytes after removing SeqTsHeader.
- seq_num: Sequence number from SeqTsHeader.
- pkt_uid: ns-3 packet UID for this packet.
- tx_time_us: Transmit timestamp from SeqTsHeader in microseconds.
- delay_us: One-way delay in microseconds (now - tx_time_us).

columns_used = [time_us, direction, rnti, pkt_size, delay_us]

rtt_trace.txt (recorded at UE echo client; RTT, no UL/DL split)
- time_us: Receive time in microseconds at the UdpEchoClient.
- ue_id: UE index for the echo client.
- imsi: IMSI for the UE.
- cell_id: Serving cell ID for the UE.
- rnti: UE RNTI from the UE RRC (if available).
- pkt_size: Packet size in bytes after removing SeqTsHeader.
- seq_num: Sequence number from SeqTsHeader.
- pkt_uid: ns-3 packet UID for this packet.
- tx_time_us: Transmit timestamp from SeqTsHeader in microseconds.
- delay_us: Round-trip time in microseconds (now - tx_time_us).

columns_used = [time_us, rnti, pkt_size, delay_us]

vrFragment_trace.txt (recorded at BurstSink (UE for DL, remote host for UL); UL or DL depends on sink placement)
- time_us: Receive time in microseconds at the BurstSink.
- ue_id: UE index inferred from the sink node or source address.
- imsi: IMSI for the UE related to this fragment.
- cell_id: Serving cell ID for that UE at log time.
- rnti: UE RNTI from the UE RRC (if available).
- burst_seq: Burst sequence number from SeqTsSizeFragHeader.
- burst_size: Total burst size in bytes from SeqTsSizeFragHeader.
- num_frags: Total fragments for the burst from SeqTsSizeFragHeader.
- frag_seq: Fragment sequence number within the burst from SeqTsSizeFragHeader.
- tx_time_us: Fragment transmit timestamp in microseconds from SeqTsSizeFragHeader.
- delay_us: One-way fragment delay in microseconds (now - tx_time_us).

columns_used = [time_us, rnti, burst_size, delay_us]

vrBurst_trace.txt (recorded at BurstSink (UE for DL, remote host for UL); UL or DL depends on sink placement)

- bwp_id: BWP ID of the gNB MAC that handled the BSR.

- subframe: Subframe number from the gNB SFN/slot.

- lcg: Logical channel group index reported in the BSR.
- bsr_level: BSR level index reported by the UE MAC.
- queue_bytes: Buffer size in bytes derived from bsr_level.

columns_used = [time_us, rnti, bsr_level, queue_bytes]

columns_used = [time_us, rnti, msg_type, rv, mcs, tb_size, num_prbs]

DlRxTbTrace.txt (recorded at UE PHY; DL)
- time_us: Simulation time in microseconds when the TB is decoded.
- frame: Frame number for the TB reception.
- subframe: Subframe number for the TB reception.
- slot: Slot number for the TB reception.
- sym_start: Starting OFDM symbol for the TB.
- num_symbols: Number of OFDM symbols for the TB.
- cell_id: Cell ID for the TB.
- bwp_id: BWP ID for the TB.
- rnti: UE RNTI for the TB.
- tb_size: Transport block size in bytes.
- mcs: MCS index for the TB.
- rank: MIMO rank for the TB.
- rv: Redundancy version for the TB.
- sinr_db: SINR in dB for the TB decoding.
- cqi: CQI reported/used for the TB.
- corrupt: 1 if TB decoding failed, 0 if successful.
- tbler: TB error rate estimate for the TB.

columns_used = [time_us, rnti, sinr_db, cqi, tbler]

DlRxTbComponentTrace.txt (recorded at UE PHY; DL)
- time_us: Simulation time in microseconds when the TB component row is logged.
- rx_pdu_id: Monotonic ID for the received PHY packet/PDU attempt (group key for component rows).
- frame, subframe, slot, sym_start, num_symbols: PHY timing/location of the received TB.
- cell_id, bwp_id, rnti: Serving identifiers for this TB.
- lcid: Logical channel ID from the radio bearer packet tag.
- tb_size: Expected/scheduled TB size (bytes) from PHY expected-TB context.
- rx_pdu_bytes: Actual received packet size for this TB attempt (bytes).
- mcs, rank, rv: PHY/MAC transmission parameters for this TB attempt.
- sinr_db, cqi, corrupt, tbler: Decoding quality/result fields for this TB attempt.
- pkt_id: Per-node pkt_id carried by Ipv4IdTag ByteTag (0 if unavailable).
- component_bytes: Number of bytes in this pkt_id component within rx_pdu_bytes.

columns_used = [time_us, rx_pdu_id, rnti, lcid, tb_size, rx_pdu_bytes, rv, corrupt, pkt_id,
component_bytes]
